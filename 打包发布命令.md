# dwsal CLI 打包发布操作手册

> 

---

## 步骤一：修改 npm 版本号

编辑文件：`Jusoft.DingTalk.MCP.CLI\npm-wrapper\package.json`

将 `"version"` 字段改为新版本号（与上表 `VERSION` 一致）：

```json
"version": "1.1.0"
```



---

## 一键发布（步骤二～六合并）

> 执行前请确认：已完成**步骤一**（修改 package.json 版本号），并将下方 `VERSION`、`OLD_VERSION`、`RELEASE_NOTES` 替换为实际值。

```powershell
$VERSION      = "1.1.0"
$OLD_VERSION  = "1.0.9"
$RELEASE_NOTES = "更新说明：修复注入的BUG"
$ROOT = "e:\工作文件\鸿欢团队资源管理项目\2025年\云效代码\product-public-module"
$CLI  = "$ROOT\Jusoft.DingTalk.MCP.CLI"

# 步骤二：多平台编译打包
dotnet publish "$CLI" -r win-x64   -c Release --self-contained true -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true -o "$CLI\publish\win-x64"
dotnet publish "$CLI" -r linux-x64 -c Release --self-contained true -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true -o "$CLI\publish\linux-x64"
dotnet publish "$CLI" -r osx-arm64 -c Release --self-contained true -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true -o "$CLI\publish\osx-arm64"
dotnet publish "$CLI" -r osx-x64   -c Release --self-contained true -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true -o "$CLI\publish\osx-x64"

# 步骤三：推送代码到 GitHub
cd $ROOT
git push origin master

# 步骤四：发布到 npm
cd "$CLI\npm-wrapper"
npm publish --access public

# 步骤五：创建 GitHub Release 并上传全平台 zip
gh release create "v$VERSION" `
  "$CLI\publish\win-x64\dwsal-win-x64.zip" `
  "$CLI\publish\linux-x64\dwsal-linux-x64.zip" `
  "$CLI\publish\osx-arm64\dwsal-osx-arm64.zip" `
  "$CLI\publish\osx-x64\dwsal-osx-x64.zip" `
  --repo QiChangWei/-CLI `
  --title "v$VERSION" `
  --notes $RELEASE_NOTES

# 步骤六：废弃旧版 npm 包
npm deprecate "dwsal@$OLD_VERSION" "已废弃，请升级至最新版本: npm install -g dwsal@latest"
```
